# Hero vs Evil Wizard - Python Turn-Based Game

A mini Python project for learning **Object-Oriented Programming (OOP)**. Players create a hero and battle the Evil Wizard in a turn-based game.

## Features
- Four hero classes: Warrior, Mage, Archer, Paladin
- Each hero has two unique abilities
- Randomized attack damage
- Healing mechanic
- Turn-based battle system
- Enemy regeneration and attacks
- Victory and defeat messages

## How to Play
1. Clone or download the repository.
2. Make sure Python 3.x is installed.
3. Run the game:
   ```bash
   python hero_battle.py
